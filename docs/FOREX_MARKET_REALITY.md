# Forex Market Reality

This repository is built for Forex analysis, but a real Forex bot needs more than chart logic.

## Important Facts

- Spot Forex is decentralized. Broker prices and spreads can differ.
- Tick volume is broker-specific. It can be useful as a proxy, but it is not centralized exchange volume.
- Leverage increases both gains and losses.
- Spreads and slippage are strategy-critical, especially during news, rollover, and low-liquidity sessions.
- SMC concepts are not standardized academic formulas. This repository turns common SMC ideas into explicit, testable rules.
- Technical indicators are deterministic formulas, but deterministic does not mean predictive.
